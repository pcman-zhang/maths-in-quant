## 项目概述

本项目面向**仅有高中数学基础**的读者，致力于系统性地讲解量化金融各个阶段所必需的数学知识。我们深知，量化金融的门槛看似被各种"低代码"工具和 AI 助手拉低了，但实际上，真正的量化投资是一门融合了数学、统计学、计算机科学和金融学的交叉学科，其根基始终扎在扎实的数学土壤之中。

因此，我们的教学设计遵循一条清晰的主线：**从高中数学出发，逐步攀登至量化金融所需的高等数学、概率统计、线性代数、优化理论以及随机分析等核心领域**。我们不追求面面俱到，而是聚焦于"做量化真正需要用到什么数学"这一核心问题，剔除冗余，直击本质。

需要特别说明的是: **本教程是一本数学教材, 而非量化交易系统教材.** 我不会教你如何搭建一套高频交易引擎, 不会讨论最新的 GPU 加速回测架构, 不会深入订单簿的微观结构, 也不会告诉你哪个因子"现在还能用". 这些都是快速变化、高度工程化、与机构具体基础设施深度绑定的领域——它们不适合写成一本面向高中数学基础的通用教材.

**本教程唯一的目标是: 让你能够从数学的角度理解量化过程.** 当你读完本书后, 看到一份量化策略的研报或风险报告, 你不会被里面的公式吓退——你会知道那个 $\Sigma$ 是协方差矩阵, 知道 $\beta$ 是对市场风险的暴露, 知道 IC 是衡量预测能力的指标, 知道那个优化问题在求解什么. 你不会成为一个"调参侠"——你会成为一个**理解自己在做什么的量化实践者**.

数学是量化大厦的地基. 工程工具会过时, 因子会失效, 市场结构会变化——但数学不会. 方差永远是 $E[(X-\mu)^2]$, 贝叶斯公式永远是 $P(H|D) \propto P(D|H)P(H)$, 无论你用的是 2024 年的 PyTorch 还是 2034 年的量子计算机.

### 章节结构：四位一体的学习闭环

为了确保读者不仅"听懂"而且"会用"，每个章节均采用以下四位一体的教学设计：

1. **数学原理**：从最直观的直觉出发，用通俗的语言和几何图像解释数学概念的来龙去脉。我们拒绝"从天而降"的公式堆砌，而是致力于讲清楚"这个公式为什么长这样"、"它解决了什么问题"。

2. **核心公式**：在理解原理的基础上，提炼出该章节最具代表性的数学公式，并详细标注每个符号的含义、适用条件及边界情况。这是从"理解"到"应用"的桥梁。

3. **Python 示例代码**：所有的数学概念都必须落地为可运行的代码。我们提供完整的、带有详细注释的 Python 实现，让读者能够亲手运行、修改参数、观察结果，在"做中学"中建立对数学的体感。代码风格追求清晰易懂，而非炫技式的过度封装。

4. **参考文献**：为学有余力的读者提供进一步深入学习的权威资料来源，包括经典教材、学术论文及业界实践报告，确保知识的可溯源性和可扩展性。

## 为什么写这个项目：对 "Vibe Coding" 现象的冷思考

自 AI 辅助编程（Vibe Coding）兴起以来，一个危险的幻觉正在蔓延：许多人——包括曾经的我——会产生一种错觉，认为自己"突然行了"。毕竟，当你可以用自然语言描述需求，AI 就能生成看似专业的代码时，那种"我也能做量化"的自信确实令人陶醉。

然而，我们必须清醒地认识到：**Vibe Coding 绝非银弹**。做量化是一个极其**严肃**的事情。金融市场不会因为你使用了 AI 生成代码就对你网开一面，它只会用真金白银的亏损来惩罚每一个基础不牢的投机者。

我可以不客气地说：**如果你不好好学量化的基本原理，不真正理解策略背后的数学逻辑和统计假设，那么在这个市场上，你必败无疑。** AI 可以帮你写代码，但它无法替你承担对数学原理的理解责任；它可以帮你调试程序，但它无法替你判断一个策略是否过拟合、是否具有统计显著性、是否满足风险约束。当市场出现极端行情时，只有真正理解自己模型底层逻辑的人，才知道该坚持还是该止损。

因此，本项目的宗旨绝非培养"调参侠"或"代码搬运工"，而是进行**量化金融数学的系统性教学**——这也是给我自己的教学。我们将结合量化投资的各个经典方法（从基础的均值-方差模型到因子投资，从时间序列分析到衍生品定价），带领读者一步步地练习、推导、编码和验证。我们的目标是：让每一位读者都能够真正掌握量化各个阶段所需的数学工具和基本逻辑，建立起"从数学原理到代码实现"的完整认知链条。

## 快速开始: 在 Jupyter Notebook 中阅读

本教程**强烈建议在 Jupyter Notebook 中阅读**. GitHub 的 Markdown 渲染与 Jupyter 存在方言差异, 部分 LaTeX 公式可能显示异常; 而 Notebook 环境可以直接运行代码、修改参数、观察结果——这正是本项目"代码即练习"理念的完整呈现.

### 1. 搭建环境 (首次使用, 约 3-5 分钟)

要求 Python 3.12+, 使用 Conda 环境 (部分数学库在 venv 下需要重新编译, Conda 下可直接安装).

```bash
# 创建并激活环境
conda env create -f environment.yaml
conda activate maths-in-quant

# 注册 Jupyter 内核 (如需在 JupyterLab / VS Code 中使用)
python -m ipykernel install --user --name=maths-in-quant --display-name "Python (maths-in-quant)"

# 验证安装
python -c "import numpy, pandas, scipy, matplotlib, statsmodels, cvxpy, sklearn, sympy, arch; print('All OK')"
```

### 2. 生成 Notebook 文件

教程源文件为 Markdown (`.md`), 一键转换为 `.ipynb`:

```bash
# 安装 jupytext (一次性)
pip install jupytext

# 批量转换
jupytext --to notebook chapters/*.md
```

### 3. 启动阅读

```bash
jupyter notebook chapters/
```
使用notebook需要先安装字体：[字体安装](#字体安装)

在打开的浏览器中选择任意章节开始. 每个代码块都可以独立运行——修改参数, 观察输出变化, 在"做中学"中建立数学体感.

### 第二阶段：概率统计——量化的核心语言

> **目标**：建立"用概率描述不确定性"的思维框架。这是量化金融中最重要的语言，没有之一。

#### [第7章 概率论基础——从掷骰子到资产收益](./chapters/chapter07.md)
- **数学原理**：样本空间、事件、概率公理；条件概率与贝叶斯思维（"看到新数据后如何更新信念"）
- **核心公式**：全概率公式、贝叶斯公式
- **Python 示例**：模拟掷骰子与股票涨跌；用贝叶斯更新判断策略是否失效
- **参考文献**：《概率论与数理统计》（茆诗松）

#### [第8章 随机变量与分布——描述不确定性的工具箱](./chapters/chapter08.md)
- **数学原理**：离散 vs 连续随机变量；金融中常见的分布：正态分布、对数正态分布、t 分布、泊松分布（跳跃过程）
- **核心公式**：PDF/CDF 定义；正态分布密度函数
- **Python 示例**：绘制收益率的直方图并拟合各种分布；检验"收益率是否服从正态分布"（厚尾现象的直观）
- **参考文献**：《统计套利》

#### [第9章 期望、方差与协方差——风险的度量语言](./chapters/chapter09.md)
- **数学原理**：建立用一阶矩和二阶矩描述"风险-收益"的完整语言体系；组合风险的数学结构 $w^T\Sigma w$；分散化的数学本质与边界；跟踪误差、信息比率与基本面主动管理定律；收缩估计与偏差-方差权衡
- **核心公式**：期望/方差/协方差/相关系数定义、组合方差矩阵形式、MCR/RC 风险分解、等权组合分散化极限、信息比率与主动管理基本定律
- **Python 示例**：真实 A 股三资产的风险-收益分析；两资产组合前沿与蒙特卡洛有效前沿；分散化效应的真实数据验证；协方差矩阵收缩估计
- **参考文献**：Markowitz 1952, Grinold & Kahn 1999, Ledoit & Wolf 2004

#### [第10章 大数定律、中心极限定理与统计推断——从数据到结论](./chapters/chapter10.md)
- **数学原理**：大数定律保证估计的一致性；中心极限定理给出估计量的分布；从 CLT 到标准误与置信区间——量化"估计有多不准"；Bootstrap 绕过理论公式；Delta 方法处理非线性统计量；有效样本量修正金融数据的时间依赖
- **核心公式**：LLN 依概率收敛、CLT 依分布收敛、标准误 $\sigma/\sqrt{n}$、置信区间、Delta 方法渐近分布、ESS 公式
- **Python 示例**：沪深 300 均值收敛验证；三分布 CLT 普适性实验；Bootstrap 与块 Bootstrap 估计夏普不确定性；有效样本量的真实数据计算
- **参考文献**：Wasserman《统计学完全教程》, Efron & Tibshirani 1993, Fama 1965

#### [第11章 假设检验与 p 值——策略有效性的统计判决](./chapters/chapter11.md)
- **数学原理**：从第 10 章的置信区间到正式的"是/否"决策——原假设与备择假设的设定；p 值的真实含义与常见误解；Type I/II 错误与统计功效；多重检验问题（Data Mining Bias）与量化过拟合
- **核心公式**：t 检验统计量、p 值定义、Bonferroni 校正
- **Python 示例**：检验某因子是否真的有效（收益率均值的 t 检验）；多重检验的直观演示——100 个随机因子中总能找到几个"显著"的
- **参考文献**：Wasserman《统计学完全教程》；《金融计量经济学》

#### [第12章 回归分析——从一元到多元，因子模型的数学基石](./chapters/chapter12.md)
- **数学原理**：从统计视角使用最小二乘——关注系数的显著性、R² 的解释力、多重共线性的危害；第 15 章将用矩阵语言重新推导同一个公式，揭示 OLS 本质上是正交投影
- **核心公式**：OLS 估计量、标准误、R²、t 统计量、F 检验
- **Python 示例**：用 CAPM 回归计算股票的 Alpha 和 Beta；Fama-French 三因子回归；检验因子溢价是否显著
- **参考文献**：Fama & French 1993 原文

### 第三阶段：线性代数——多资产世界的语言

> **目标**：当投资组合从"单股票"扩展到"多资产"时，标量运算已不够用，必须掌握向量和矩阵。

#### 第13章 [向量与矩阵——多资产组合的数学表示](./chapters/chapter13.md)
- **数学原理**：向量作为"持仓权重"或"收益率序列"——加法（合并子组合）、数乘（杠杆/减仓）、点积（组合收益率 = w·r）；范数与余弦相似度的几何直觉；矩阵的"列视角"（每列=一只资产的收益率时间序列）；矩阵-向量乘法 R@w 一次性计算全时期组合收益
- **核心公式**：向量加减、数乘、点积、范数、余弦相似度、矩阵-向量乘法、线性组合
- **Python 示例**：代码融入各小节——创建等权/主动/多空权重向量；点积计算单日组合收益；矩阵-向量乘法批量计算全时期净值曲线；6只股票余弦相似度矩阵可视化
- **参考文献**：Lay《线性代数及其应用》, Strang《Introduction to Linear Algebra》, Markowitz 1952

#### 第14章 [矩阵运算与特征分解——协方差矩阵的密码](./chapters/chapter14.md)
- **数学原理**：矩阵乘法 R@W 批量回测 K 个策略；因子收益归因 F@B^T；逆矩阵与最小方差组合 w = Σ^{-1}1 / (1^T Σ^{-1} 1)；Ledoit-Wolf 收缩估计（行业标准）；特征值/特征向量从零开始手算推导（det(A-λI)=0 → λ → v → 验证）；特征值=风险方向方差；PCA 风险分解（碎石图、累积解释比）；Eigenportfolio 与统计套利
- **核心公式**：矩阵乘法、转置性质、逆矩阵、特征方程 Av=λv、特征分解 Σ=QΛQ^T、Ledoit-Wolf 收缩、条件数 κ=λmax/λmin
- **Python 示例**：代码融入各小节——5策略批量回测（R@W）；sklearn Ledoit-Wolf 收缩（条件数 519→233）；50只A股 PCA 风险分解（PC1=45.6%, 13个PC解释80%）；最小方差组合样本外对比（波动率 17.4%→13.5%, 夏普 0.36→0.96）
- **参考文献**：Lay, Strang, Ledoit & Wolf 2004, Meucci 2005, Avellaneda & Lee 2010

#### [第15章 线性方程组与最小二乘法——求解最优参数](./chapters/chapter15.md)
- **数学原理**：线性方程组 Ax=b 的三种命运（适定/超定/欠定）及其金融对应（复制组合/回归/正则化）；矩阵求导基础与正规方程推导（令梯度为零 → X^T X β = X^T y → β̂ = (X^T X)^{-1} X^T y）；投影矩阵 P = X(X^T X)^{-1} X^T 的几何本质——ŷ = Py 是 y 在 X 列空间上的正交投影，残差正交（X^T ε̂ = 0），勾股定理成立（||y||² = ||ŷ||² + ||ε̂||²）；CAPM Beta = 投影系数，R² = cos²θ
- **核心公式**：正规方程、OLS 矩阵解、投影矩阵 P、幂等性 P²=P、残差正交、R² 几何解释
- **Python 示例**：纯矩阵法 OLS（手写 (X^T X)^{-1} X^T y）与 statsmodels 对比验证；3D 投影可视化展示 y = ŷ + ε̂ 的正交分解
- **参考文献**：Strang, Hastie et al. ESL, Grinold & Kahn, Magnus & Neudecker

### 第四阶段：优化理论——投资决策的数学

> **目标**：学会用数学语言表述"在约束条件下最大化收益/最小化风险"。

#### 第16章 多元函数优化——从梯度到海森矩阵
- **数学原理**：从一元导数（第3章）走向多元梯度——方向导数、梯度向量、海森矩阵；凸性的二阶判定（海森矩阵正半定）；梯度下降法的几何直觉（"沿着最陡的方向下山"）；牛顿法的二次收敛
- **核心公式**：梯度 $\nabla f$、海森矩阵 $\mathbf{H}$、梯度下降迭代 $x_{k+1} = x_k - \eta \nabla f(x_k)$、牛顿迭代
- **Python 示例**：用梯度下降拟合线性回归参数；可视化梯度下降路径与等高线；对比梯度下降与牛顿法的收敛速度
- **参考文献**：Boyd《凸优化》第1-2, 9章

#### 第17章 带约束优化与拉格朗日乘子——投资组合优化的核心
- **数学原理**：等式约束的几何意义（在曲面上寻找极值）；拉格朗日乘子的金融解释（"风险的影子价格"）
- **核心公式**：拉格朗日函数、KKT 条件（简单版本）
- **Python 示例**：解析求解两资产均值-方差最优组合；推导有效前沿的数学表达式
- **参考文献**：Markowitz 1952 原文、Merton 1972

#### 第18章 凸优化与数值方法——当解析解不存在时
- **数学原理**：凸集与凸函数；为什么金融优化问题喜欢"凸"的（局部最优=全局最优）；二次规划（QP）的框架
- **核心公式**：二次规划标准形式
- **Python 示例**：用 `cvxpy` 库求解带约束（无卖空、单资产权重上限）的 Markowitz 优化问题
- **参考文献**：Boyd《凸优化》

### 第五阶段：时间序列分析——动态世界的建模

> **目标**：金融数据是带时间戳的序列，必须学会建模"今天与昨天的关系"。

#### 第19章 时间序列基础——平稳性与自相关
- **数学原理**：平稳性的直觉（统计特性不随时间改变）；自相关函数（ACF）与偏自相关函数（PACF）
- **核心公式**：自协方差、自相关系数、Ljung-Box 检验
- **Python 示例**：检验股票价格序列是否平稳；绘制 ACF/PACF 图
- **参考文献**：Box & Jenkins《时间序列分析》

#### 第20章 移动平均与自回归模型——ARIMA 入门
- **数学原理**：用过去预测现在；AR(p)、MA(q) 的直觉；ARIMA 的差分操作（将非平稳序列变平稳）
- **核心公式**：AR(1)、MA(1)、ARIMA(p,d,q) 公式
- **Python 示例**：用 `statsmodels` 拟合 ARIMA 模型；预测未来收益率（并讨论预测局限性）
- **参考文献**：《金融时间序列分析》（Tsay）

#### 第21章 GARCH 模型——波动率的聚类现象
- **数学原理**：金融序列的"波动聚集"特征（大波动后面跟着大波动）；GARCH(1,1) 的递归结构
- **核心公式**：GARCH(1,1) 均值方程与方差方程
- **Python 示例**：用 `arch` 库拟合 GARCH 模型；预测未来波动率；计算 VaR（风险价值）
- **参考文献**：Engle 1982, Bollerslev 1986 原文

#### 第22章 协整与配对交易——非平稳序列的长期关系
- **数学原理**：伪回归问题（为什么不能用普通回归处理非平稳序列）；单位根检验（ADF）的数学原理；协整的严格定义与 Johansen 检验框架；误差修正模型（VECM）的推导——短期波动如何被长期均衡"拉回"
- **核心公式**：ADF 检验统计量、Johansen 迹检验与最大特征值检验、误差修正方程 $\Delta y_t = \alpha \beta^T y_{t-1} + \Gamma \Delta y_{t-1} + \varepsilon_t$
- **Python 示例**：ADF 检验验证股票对数价格的单位根；Johansen 检验识别协整股票对；构建 VECM 模型估计误差修正速度；设计均值回归信号并评估样本外表现
- **参考文献**：Engle & Granger 1987, Johansen 1991, Vidyamurthy《配对交易》

### 第六阶段：随机分析——衍生品定价的门槛

> **目标**：理解衍生品定价的数学基础。这是从"统计套利"迈向"定价理论"的关键一步。

#### 第23章 从随机游走到布朗运动——随机过程的直觉

- **核心公式**：布朗运动定义、伊藤等距（简单介绍）

#### 第24章 伊藤引理——随机微积分的核心
- **数学原理**：为什么普通微积分在随机世界失效（布朗运动路径无限粗糙）；伊藤引理的直觉（二阶项不可忽略）
- **核心公式**：伊藤引理（一元与多元版本）

#### 第25章 Black-Scholes 模型——期权定价的里程碑
- **数学原理**：风险中性定价的直觉（"在虚拟世界里，所有资产的期望收益率等于无风险利率"）；BS PDE 的推导思路
- **核心公式**：Black-Scholes 欧式看涨/看跌期权定价公式；Greeks 公式

#### 第26章 蒙特卡洛模拟与数值方法——当解析解不存在时
- **数学原理**：大数定律在模拟中的应用；路径依赖型衍生品（亚式期权、回望期权）的定价思路；方差缩减技术（对偶变量、控制变量）
- **核心公式**：风险中性定价的一般形式、蒙特卡洛估计量
- **Python 示例**：用蒙特卡洛模拟定价亚式期权；实现方差缩减技术并对比效率

### 第七阶段：因子模型与机器学习——从线性到非线性的量化前沿

> **目标**：将经典统计和线性代数工具整合为完整的因子研究框架，并扩展到高维、非线性的现代量化场景。

#### 第27章 多因子模型——从因子构造到组合构建
- **数学原理**：将第12章的回归和第14章的协方差矩阵整合为量化股票投资的完整框架——因子构造（排序法、打分法、中性化处理）；因子检验（IC/Rank IC、IR、分层回测、Fama-MacBeth 两步法）；因子组合（等权、IC 加权、信息比最优化）；Alpha 模型与风险模型的分工
- **核心公式**：IC 定义与标准误、IR = IC × √breadth、Fama-MacBeth 横截面回归、Barra 风格归因
- **Python 示例**：构建一个完整的单因子（如动量）研究流程——信号生成、中性化、IC 序列分析、分层回测；对比等权组合与最优组合的绩效差异；用 Barra 模型做绩效归因
- **参考文献**：Grinold & Kahn 1999, Fama & French 1993, Fama & MacBeth 1973

#### 第28章 监督学习与正则化——从逻辑回归到高维因子选择
- **数学原理**：从线性分类（逻辑回归）到非线性边界（SVM、决策树）；正则化的贝叶斯解释（L1=拉普拉斯先验，L2=高斯先验）；Lasso 的变量选择功能与因子筛选的天然契合；偏差-方差权衡与交叉验证的数学基础
- **核心公式**：Sigmoid 函数、交叉熵损失、Lasso/Ridge/Elastic Net 目标函数、K-Fold CV 估计量
- **Python 示例**：用逻辑回归+Lasso 从候选因子池中筛选有效因子；用 Ridge 回归改善高维协方差矩阵的条件数（与第14章的收缩估计呼应）；交叉验证选择最优正则化强度
- **参考文献**：《统计学习基础》（Hastie, Tibshirani, Friedman）第3-4, 7章

#### 第29章 聚类分析——市场状态的识别与资产分组
- **数学原理**：距离度量与相似度矩阵；K-Means 的迭代优化（Lloyd 算法）与收敛性；层次聚类的连接准则（Ward、Complete、Average）；聚类在量化中的三类应用——行业分类的纯数据驱动替代、市场状态（牛市/熊市/震荡）的自动识别、组合分层构建（Cluster Risk Parity）
- **核心公式**：K-Means 目标函数、Ward 距离递推公式、轮廓系数（Silhouette Score）
- **Python 示例**：对 50 只 A 股按收益率特征聚类，与真实行业分类对比一致性（调整兰德指数）；按波动率/相关性特征聚类识别市场状态；用聚类结果构建分层风险平价组合
- **参考文献**：Hastie et al. 第14章、《数据挖掘：概念与技术》

## 章节编排的"隐藏逻辑"

1. **先定义后推断**：先讲风险的描述语言——期望/方差/协方差（第 9 章），再讲估计不确定性的量化——LLN/CLT/置信区间/Bootstrap（第 10 章），最后进入正式的假设检验（第 11 章）。三层递进：风险是什么 → 量的有多准 → 怎么正式判决
2. **统计视角 → 几何视角**：第 12 章用微积分和统计直觉讲回归（最小化残差平方和、t检验、R²），第 15 章用线性代数重新推导同一个公式（正规方程、投影矩阵、正交分解）。两条路径殊途同归，让读者看到数学的"对偶性"——同一个事实，两种语言
3. **PCA 只讲一次**：特征值分解和 PCA 在第 14 章（线性代数阶段）完整讲授——从手算 2×2 矩阵的特征值到 50 只 A 股的 PCA 风险分解。后续第 28 章不再重复 PCA 基础，而是聚焦正则化方法（Lasso/Ridge/Elastic Net）在高维因子选择中的应用
4. **先静后动**：先讲截面数据（某一时刻的多个资产），再讲时间序列（单一资产的历史演变）
5. **先解析后数值**：每个阶段都是先给解析解/闭式解，再给数值方法（蒙特卡洛、梯度下降），培养"能推公式、也能写代码"的双重能力
