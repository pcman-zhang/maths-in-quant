## D.1 各章练习题索引

### 第7章 概率论基础

**数学推导**:

1. 容斥原理推广——证明 $P(A \cup B \cup C)$ 的三事件形式
2. 条件概率与 odds——证明贝叶斯公式的 odds 形式: 后验 odds = 似然比 × 先验 odds
3. 全概率与期望——证明 $E[R] = \sum E[R \mid B_i] P(B_i)$ 与全概率公式的联系

**编程实践**:

4. 蒙特卡洛验证贝叶斯——模拟策略状态和交易结果, 验证后验概率
5. 多策略贝叶斯比较——三个策略的后验概率模型选择
6. 真实数据贝叶斯仪表盘——滚动计算后验概率 + 动态仓位建议

### 第8章 随机变量与分布

**数学推导**:

1. 泊松分布 PMF → CDF → 多期概率(异常月份的超几何/二项计算)
2. 正态分布的线性组合 → VaR → 相关性对 VaR 的影响
3. 对数正态分布的期望推导(矩母函数法) + $\sigma^2/2$ 修正的金融含义

**编程实践**:

4. VaR 敏感性分析——正态 vs t 分布, $\nu$ 参数的影响曲线
5. 蒙特卡洛验证厚尾——1 亿仓位在一年内被低估多少次?

### 第9章 期望、方差与协方差

**数学推导**:

1. 期望的线性性——组合期望收益不依赖相关性
2. 最优权重推导——$w_A^* = (\sigma_B^2 - \sigma_{AB}) / (\sigma_A^2 + \sigma_B^2 - 2\sigma_{AB})$
3. 分散化的数学极限——$\lim_{n\to\infty} \sigma_p^2 = \rho\sigma^2$

**编程实践**:

4. 协方差矩阵的估计窗口敏感性——120 天 vs 723 天, VaR 差异
5. 滚动相关系数与分散化时机——波动加剧时相关性上升还是下降?

## D.2 大作业: 从零构建一个多因子选股策略

### 目标

独立完成一个**可运行, 可解释, 可通过基本回测检验**的多因子选股策略. 这个项目将串联你在本书中学到的几乎所有核心技能:

| 阶段 | 用到的章节 | 核心任务 |
|------|-----------|---------|
| 1. 数据准备 | Ch5, 附录C | 获取并清洗价格数据, 计算对数收益率 |
| 2. 因子构造 | Ch2-6, Ch9 | 构造至少 3 个因子(估值/动量/质量等) |
| 3. 因子检验 | Ch7-8, Ch11-12 | IC 分析, 分层回测, 显著性检验, 分布检验 |
| 4. 组合构建 | Ch9, Ch16-18 | 因子合成, 权重优化(均值-方差或等权) |
| 5. 回测评估 | Ch9-11 | 计算夏普/最大回撤/换手率, 与基准对比 |
| 6. 报告撰写 | — | 总结策略逻辑, 绩效归因, 风险提示 |

### 阶段 1: 数据准备

**输入**: 从 AKShare 或 Yahoo Finance 获取沪深 300 成分股的日线数据(至少 50 只, 3 年以上).

**输出**: 一个干净的 `DataFrame`, 行=日期, 列=股票代码, 值=收盘价(或对数收益率).

**关键检查点**:
- [ ] 处理停牌(前向填充? 剔除?)
- [ ] 处理除权除息(使用后复权价格)
- [ ] 确保没有前视偏差(计算因子时用到的数据不能包含未来信息)

### 阶段 2: 因子构造

构造至少 3 个因子, 每个因子应该有清晰的**经济直觉**:

| 因子类别 | 建议因子 | 计算公式 | 简要逻辑 |
|---------|---------|---------|---------|
| 估值 | 市净率倒数(BP) | 净资产 / 总市值 | 便宜的公司长期收益更高 |
| 动量 | 过去 12-1 个月收益 | $\prod_{t=12}^{1} (1+R_{t}) - 1$ | 趋势延续效应 |
| 低波动 | 过去 60 日波动率的倒数 | $1 / \sigma_{60d}$ | 低波动异象 |
| 质量 | ROE | 净利润 / 净资产 | 高盈利能力的公司溢价 |

**关键检查点**:
- [ ] 每个因子做行业中性化(在行业内部做 Z-score 标准化)
- [ ] 处理极端值(Winsorize 到 1% 和 99% 分位数)
- [ ] 因子值做 Z-score 标准化(均值 0, 标准差 1)

### 阶段 3: 因子检验

对每个因子单独做以下检验:

**IC 分析**(第 9 章):
- 计算 Rank IC(因子值与下期收益率的 Spearman 相关系数)
- 计算 IC 均值, IC 标准差, ICIR(IC 的信息比率)
- ICIR > 0.3 通常被认为是可用的因子

**分层回测**(第 11 章):
- 按因子值将股票分成 5 组(Q1=最小, Q5=最大)
- 每组做等权组合, 计算每组的年化收益和夏普比率
- Top-Bottom(Q5 - Q1)组合应呈现单调的收益差异

**显著性检验**(第 11 章):
- Top-Bottom 组合的收益是否显著不等于 0?(t 检验)
- 考虑多重检验问题(你测试了多个因子)

**分布检验**(第 8 章):
- 因子收益的分布是否有厚尾? 偏度如何?

### 阶段 4: 组合构建

**因子合成**(第 9 章):
- 将多个因子的 Z-score 等权或 IC 加权合成为一个综合得分
- 确保因子之间没有严重共线性(两两相关系数 > 0.7 需要合并或剔除)

**选股**: 选择综合得分最高的 20-30 只股票.

**权重分配**(第 16-18 章):
- 方案 A(简单): 等权
- 方案 B(进阶): 均值-方差优化(`cvxpy` 或 `scipy.optimize`)
- 方案 C(风控): 风险平价(每个股票对组合风险的贡献相等)

**约束**:
- 单只股票权重上限 10%
- 行业集中度上限 30%
- 月换手率上限 50%(单边)

### 阶段 5: 回测评估

**回测周期**: 样本外至少 1 年(保留最后 20% 数据不做任何训练).

**绩效指标**:

| 指标 | 公式 | 目标 |
|------|------|------|
| 年化收益 | $\mu \times 252$ | > 基准(沪深300) |
| 年化波动率 | $\sigma \times \sqrt{252}$ | < 25% |
| 夏普比率 | $(\mu_p - r_f) / \sigma_p$ | > 0.5 |
| 最大回撤 | $\max(\text{峰值} - \text{谷值}) / \text{峰值}$ | < 30% |
| 信息比率 | $(\mu_p - \mu_b) / \sigma_{p-b}$ | > 0.5 |
| 月胜率 | 月度超额收益 > 0 的月份占比 | > 55% |
| 换手率 | 月度双边换手 | < 100% |

**与基准对比**:
- 累计收益曲线(策略 vs 沪深 300)
- 滚动 12 个月超额收益
- 在不同市场环境(牛/熊/震荡)下的表现分解

### 阶段 6: 报告撰写

一份专业的策略报告至少应包含:

1. **策略逻辑**(1 页): 为什么这些因子应该有效? 经济直觉是什么?
2. **因子分析**(1-2 页): IC 统计, 分层回测结果, 因子相关性矩阵
3. **组合特征**(1 页): 持仓数量, 行业分布, 市值分布, 换手率
4. **绩效分析**(1-2 页): 核心指标表, 收益曲线图, 回撤图, 滚动 Sharpe
5. **风险提示**(半页): 策略的主要风险来源, 可能在什么环境下失效
6. **改进方向**(半页): 如果给你更多时间, 你会改进什么?

### 评分标准(自学参考)

| 维度 | 权重 | 优秀标准 |
|------|------|---------|
| 代码质量 | 20% | 模块化, 可复现, 有注释 |
| 因子逻辑 | 20% | 每个因子有清晰的经济直觉, 而非纯数据挖掘 |
| 统计严谨性 | 25% | ICIR, t 检验, 多重检验修正, 过拟合防范 |
| 回测合规性 | 20% | 样本外验证, 无前视偏差, 考虑交易成本 |
| 报告质量 | 15% | 逻辑清晰, 图标准确, 风险诚实, 不自欺欺人 |

## D.3 进阶挑战(可选)

完成大作业后, 可以考虑以下进阶方向:

1. **引入机器学习**: 用随机森林或 XGBoost 替代线性因子合成(第 27 章)
2. **动态因子权重**: 根据市场状态(第 8 章 GMM 识别)动态调整因子权重
3. **交易成本建模**: 加入市场冲击模型, 评估策略的真实容量
4. **风险模型升级**: 用 Barra 结构化风险模型替代样本协方差矩阵(第 14 章)
5. **另类数据**: 引入新闻情绪或分析师预期数据作为额外因子

---
